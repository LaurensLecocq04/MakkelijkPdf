## 🚀 Installatie

1. **Python 3.7+** vereist
2. **Installeer dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Installeer poppler** (zie README.md voor details)
4. **Start applicatie:**
   ```bash
   python main.py
   ```
